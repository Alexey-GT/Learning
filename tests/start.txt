Папка:
mkdir environments
cd environments

Создание окружения:
python -m venv selenium_env

Активация окружения:
selenium_env\Scripts\activate.bat

Выход из окружения:
deactivate

Запуск интерпретатора:
python

Выход из интерпретатора:
exit()

Установка Selenium:
pip install selenium==4.*

Проверка selenium:
pip list
